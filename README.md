🎓 Grade Calculator

A beautiful Grade Calculator built using Python and Streamlit. It calculates percentage and grade based on marks entered for 5 subjects.

🔗 Live App:
https://grade-calculator-pnhc7csbdt4c2pk8zfhqmq.streamlit.app/

---

✨ Features

- 👩‍🎓 Student Name Input
- 📚 5 Subject Marks Entry
- 📊 Total Marks Calculation
- 🎯 Percentage Calculation
- 🏆 Grade Calculation
- ⭐ Achievement Badges & Stars
- 🎨 Beautiful Sky Blue & Baby Pink UI
- 📈 Progress Bar
- 📱 Responsive Design

---

🛠️ Technologies Used

- Python
- Streamlit

⚙️ Installation

1] Clone Repository:

https://github.com/meshramvanshika33-rgb/grade-calculator.git

2] Navigate to the project directory:

cd grade_calculator

3] Install Requirements:

pip install -r requirements.txt

4] Run App:

streamlit run app.py
